In order to rebuild the solution to CheckersCLI you must:

	Double click the Visual C# Project file 'CheckersCLI' in the Source 'Code folder'.
	
	You can now run the debug version by pressing F5.

	If you wish to build a new release version of the system you must go to: 
	Build -> Batch Build -> Tick the check box for release version -> Press build button.

	After this your release version will be located in Source Code\obj\x86\Release
